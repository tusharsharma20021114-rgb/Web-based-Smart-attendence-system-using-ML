# 🚀 Quick Start Guide

## Get Your App Running in 5 Minutes!

### Step 1: Clone & Install (2 minutes)

```bash
git clone https://github.com/tusharsharma20021114-rgb/Web-based-Smart-attendence-system-using-ML.git
cd Web-based-Smart-attendence-system-using-ML
pip install -r requirements.txt
```

### Step 2: Start MongoDB (1 minute)

```bash
# Ubuntu/Linux
sudo systemctl start mongodb

# macOS
brew services start mongodb-community

# Windows
net start MongoDB
```

### Step 3: Run the App (30 seconds)

```bash
python3 app.py
```

### Step 4: Open Browser (30 seconds)

Navigate to: **http://localhost:5000**

### Step 5: Start Using! (1 minute)

1. **Dashboard** - View statistics
2. **Enroll** - Add students and capture images
3. **Take Attendance** - Mark attendance with camera
4. **Records** - View and export data

## 🌐 Deploy Online (Optional)

### Deploy to Render (Free)

1. Fork this repository on GitHub
2. Go to [render.com](https://render.com) and sign up
3. Click "New +" → "Web Service"
4. Connect your GitHub repository
5. Click "Create Web Service"
6. Wait 2-3 minutes for deployment
7. Your app is live! 🎉

**Your URL**: `https://your-app-name.onrender.com`

### Deploy to Railway (Free)

1. Go to [railway.app](https://railway.app)
2. Click "Start a New Project"
3. Select "Deploy from GitHub repo"
4. Choose your repository
5. Railway auto-deploys!
